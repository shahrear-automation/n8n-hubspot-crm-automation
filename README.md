# n8n HubSpot CRM Automation

This project contains an automated CRM workflow built with n8n.

The system processes incoming leads, prevents duplicate contacts in HubSpot, and automatically creates deals linked with the correct contact.

## Features

• Lead intake via webhook  
• Duplicate contact detection  
• Automatic contact upsert  
• Automated deal creation  
• Contact and deal association  

## Tools Used

- n8n
- HubSpot API
- Webhooks
- JSON data processing
